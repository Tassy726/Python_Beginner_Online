# Python初級講座

オンライン家庭教師「まなぶてらす」のPython初級講座で使用するファイルをまとめたリポジトリです。

---

## 📁 フォルダ構成

| フォルダ | 内容 |
|---|---|
| 1）レッスンファイル | 各回のレッスンで使うPythonファイル（.py）|
| 2）練習問題 | レッスン中に取り組む練習問題（.txt）|
| 3）宿題 | 各回の宿題ファイル |
| 4）演習 | 追加の演習問題 |

---

## 📋 レッスン一覧

| 回 | テーマ |
|---|---|
| 第1回 | 変数 |
| 第2回 | データ型・演算子 |
| 第3回 | カンタン計算機（input・型変換） |
| 第4回 | 条件分岐（IF文）|
| 第5回 | IF文② 比較演算子 |
| 第6回 | モジュール・おみくじアプリ |
| 第7回 | くり返し（ループ）|
| 第8回 | リスト |
| 第9回 | 関数 |
| 第10回 | tkinter（GUIアプリ入門）|

---

## 💻 ファイルのダウンロード方法

1. 画面右上の緑色の **Code** ボタンをクリック
2. **Download ZIP** を選択してダウンロード
3. ダウンロードしたZIPファイルを展開する

---

## 🛠 必要な環境

- Python 3.x
- Visual Studio Code（VSCode）
