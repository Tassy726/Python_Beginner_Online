レッスン４「条件分岐（IF文）」の練習問題


Q1）　次のプログラムの実行結果を予想してから、実行してみよう
 
hp = 0
if hp == 0:
    print("ゲームオーバー！")
 
→ hpの値を 50 に変えたらどうなる？試してみよう
→ 何も表示されなかったのはなぜ？考えてみよう
 
 
 
Q2）　Q1のプログラムに elif を追加して、
      2パターンの表示ができるようにしてみよう
 
hp = 50
 
・hpが0のとき   → 「ゲームオーバー！」
・hpが50のとき  → 「まだまだ元気！」
 
→ できたら、hpの値を 0 と 50 に変えて、
   それぞれ正しく表示されるか確認してみよう
→ hpを 999 にしたらどうなる？何が起きるか試してみよう



Q3）　Q2のプログラムに else を追加して、
      「0でも50でもない場合」にも対応させよう
 
hp = 999
 
・hpが0のとき   → 「ゲームオーバー！」
・hpが50のとき  → 「まだまだ元気！」
・それ以外のとき → 「HPは0でも50でもないよ」
 
→ hpの値を 0、50、999 に変えて、
   3パターンとも正しく表示されるか確認してみよう



Q4）　input()を使って、好きな果物を聞いて
      メッセージを変えるプログラムを作ろう
 
キーボードから果物の名前を入力してもらい、
入力された果物によって違うメッセージを表示しよう。
 
・"リンゴ"  → 「リンゴは赤いね！」
・"バナナ"  → 「バナナは黄色いね！」
・"メロン"  → 「メロンは高級だね！」
・それ以外  → 「その果物は知らないなぁ」
 
＜実行例＞
好きな果物を入力してね：バナナ
バナナは黄色いね！
 
ヒント：input()で受け取った値はそのまま文字列（str型）だよ。
        今回は型の変換は必要なし！
